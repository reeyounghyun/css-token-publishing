Alert / Banner

info/success/warning/error

Overview

Alert는 화면/섹션 안에서 상태나 주의사항을 알리는 인라인 메시지다. Banner는 화면/섹션 최상단에 전체 폭으로 걸리는 공지다 (아래 Examples 참고). 사용자 확인이 꼭 필요한 내용은 Alert 대신 Modal을 쓴다.

Variants

클래스 설명
alert 기본형 (info). 회색 아이콘
alert--primary 성공. accent 색상
alert--warning 경고. warning 색상
alert--error 에러. danger 색상
alert [alert--{primary|warning|error}]
참고
변경 사항은 저장 후 최대 5분 내에 반영됩니다.
저장 완료
모든 변경 사항이 정상적으로 저장되었습니다.
저장 용량 임박
저장 공간이 얼마 남지 않았습니다. 곧 90%에 도달합니다.
저장 실패
네트워크 오류가 발생했습니다. 잠시 후 다시 시도해주세요.

Options

클래스 설명
alert__close 선택. 생략하면 사용자가 닫을 수 없는, 계속 유지되어야 하는 안내가 된다
alert__close 생략 — 계속 유지되어야 하는 안내
닫기 버튼 없음
계속 유지되어야 하는 안내는 마크업에서 alert__close를 생략합니다.

States

별도 상태 클래스는 없다. alert__close에 data-dismiss="alert"가 있으면 js/main.js가 클릭 시 가장 가까운 .alert를 DOM에서 제거한다 — 위 Variants 데모에서 닫기 버튼을 눌러 확인할 수 있다.

Examples

banner > banner__action, banner__close — 화면 최상단 전체 폭 공지

Guidelines

✓ 권장

  • 제목은 짧게, 설명은 다음 행동까지 안내
  • 사이트 전체에 걸친 공지는 Banner, 특정 화면/폼 안 메시지는 Alert
  • 색상만으로 의미 전달하지 않기 (아이콘 + 텍스트 병행)

✕ 지양

  • 한 화면에 Alert 여러 개를 동시에 쌓아 보여주기
  • 사용자 확인이 꼭 필요한 내용 (Modal을 써야 함)
  • Banner 안에 여러 개의 액션 버튼 넣기 (하나만)
