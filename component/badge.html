Badge / Tag

색상·크기 변형, 삭제 가능한 태그

Overview

배지(Badge/Tag)는 상태·카테고리·개수 같은 짧은 정보를 눈에 띄게 표시하는 작은 라벨이다. 필터 태그처럼 사용자가 직접 삭제할 수 있는 형태로도 쓴다.

Variants

클래스 설명
badge--default 회색. 의미 없는 단순 라벨
badge--primary accent 색상. 활성/긍정적 상태
badge--error danger 색상. 위험/부정적 상태
badge--outline 테두리만, 배경 없음
badge badge--{default|primary|error|outline}
Default Active Error Outline

Options

클래스 설명
badge--sm · badge--lg 크기. 기본은 md(미지정)
badge__dot 선택. 텍스트 앞에 작은 점을 추가해 상태를 강조
badge__remove 선택. 삭제 버튼 추가. data-tag-remove 속성이 있어야 클릭 시 제거된다
badge--{sm|lg} — 크기 변형
Small Medium (기본) Large
badge__dot — 상태 점 표시
Active Invited Suspended

States

별도 상태 클래스는 없다. badge__remove에 data-tag-remove가 있으면 js/main.js가 클릭 시 가장 가까운 .badge를 DOM에서 제거한다 — 아래 Examples에서 실제로 눌러 지울 수 있다.

Examples

badge badge--outline > badge__remove(data-tag-remove) — 삭제 가능한 필터 태그

Guidelines

✓ 권장

  • 텍스트는 1~2단어로 짧게
  • 상태 의미(성공/실패 등)엔 색상 variant, 단순 라벨엔 default/outline
  • 삭제 가능한 태그는 badge--outline + badge__remove 조합으로 시각적 구분

✕ 지양

  • 배지 안에 긴 문장 넣기
  • 삭제 외의 클릭 액션이 필요한 곳에 배지를 버튼 대용으로 사용
  • 같은 색상을 프로젝트마다 다른 의미로 쓰기 (색상-의미 매핑을 일관되게)
