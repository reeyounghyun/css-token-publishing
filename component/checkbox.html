Checkbox / Radio

체크박스, 라디오

Overview

체크박스는 여러 옵션 중 원하는 만큼(0개~전부) 선택할 때, 라디오는 같은 name 그룹 안에서 정확히 하나만 선택할 때 쓴다. 클래스는 항상 checkbox/radio 고정이고, 상태는 별도 BEM 수정자 없이 네이티브 checked/disabled 속성으로만 표현한다.

Variants

클래스 설명
checkbox 여러 개 동시 선택 가능
radio 같은 name 그룹 내에서 하나만 선택 가능

Options

독립적인 수정자 클래스는 없다 — 크기/색상 변형 없이 상태(States)로만 구분된다.

States

checkbox [checked] [disabled]
radio [checked] [disabled]

Examples

이용약관 동의 — 체크박스 그룹
요금제 선택 — 라디오 그룹

Guidelines

✓ 권장

  • 독립적인 on/off는 체크박스, 상호 배타적 선택은 라디오
  • label로 감싸서 텍스트를 클릭해도 선택되게
  • 같은 라디오 그룹은 반드시 동일한 name 공유

✕ 지양

  • 선택지가 1개뿐인 체크박스 (그럴 땐 Switch가 더 적합)
  • 체크박스로 상호 배타적 선택 구현 (라디오를 써야 함)
  • disabled + checked 상태를 아무 설명 없이 방치 (왜 못 바꾸는지 안내 필요)
