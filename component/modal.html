Modal

sm/md/lg, 폼/테이블, 하단버튼 없는 버전

Overview

Modal은 현재 화면 위에 오버레이로 뜨는 대화상자다. 사용자 확인이 꼭 필요한 동작이나, 별도 페이지로 옮기기엔 가벼운 폼/상세정보를 보여줄 때 쓴다. data-modal-open으로 열고 data-modal-close·바깥(딤) 클릭·ESC로 닫힌다 — js/main.js가 전부 처리한다.

Variants

스타일 변형 없음 — modal 기본형 하나만 존재한다.

Options

클래스 설명
modal--sm / modal--lg 360px / 640px. 기본은 md 느낌의 400px(미지정)
modal__footer 선택. 생략하면 X 버튼과 바깥 클릭으로만 닫히는 모달이 된다
modal [modal--sm|modal--lg] — 트리거를 눌러서 실제로 열어볼 수 있다

공지사항

닫기(X) 또는 바깥 클릭으로만 닫을 수 있는, 확인/취소 버튼이 없는 모달입니다.

States

modal-overlay.is-open이 붙으면 화면에 나타난다. js/main.js가 트리거 클릭 시 이 클래스를 토글하고, body.modal-open으로 배경 스크롤을 잠근다. 실제 열림/닫힘은 위 Options나 아래 Examples의 버튼을 눌러 확인할 수 있다.
