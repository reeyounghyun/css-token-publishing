Accordion

<details>/<summary> 기반, JS 없이 동작

Overview

아코디언은 콘텐츠를 접고 펼쳐서 화면 공간을 절약하는 컴포넌트다. FAQ, 설정 그룹처럼 한 번에 다 볼 필요 없는 정보를 묶을 때 쓴다. 네이티브 details/summary 태그 기반이라 JS 없이 열림/닫힘이 동작한다.

Variants

스타일 변형 없음 — accordion 기본형 하나만 존재한다.

Options

속성 설명
open 페이지 로드 시 기본으로 펼쳐둘 항목에 추가 (details 네이티브 속성)
name="..." 같은 name을 가진 details끼리 묶여서, 하나를 열면 나머지는 자동으로 닫힘 (한 번에 하나만 열리는 아코디언)

States

accordion__item(details) [open] [is-disabled]
기본 (닫힘)
기본 상태에서는 닫혀 있다.
열림 (open 속성)
open 속성을 주면 페이지 로드 시 펼쳐진 상태로 시작한다.
비활성 (is-disabled)
비활성 항목입니다.

Examples

FAQ 목록 — name 속성 없음, 여러 개 동시에 열림
환불은 어떻게 받을 수 있나요?
구매 후 7일 이내에 고객센터로 문의하시면 환불 처리를 도와드립니다. 이미 사용한 크레딧은 환불 대상에서 제외됩니다.
플랜은 언제든 변경할 수 있나요?
네, 설정 > 결제 메뉴에서 언제든 상위/하위 플랜으로 변경할 수 있습니다. 변경 사항은 다음 결제 주기부터 적용됩니다.
팀원을 몇 명까지 초대할 수 있나요?
프로 플랜은 최대 10명, 엔터프라이즈 플랜은 무제한으로 팀원을 초대할 수 있습니다.
문의 준비중 (비활성)
아직 준비되지 않은 항목입니다.

단일 열림형 (name 그룹)

accordion__item(details) name="grouped" — 하나 열면 나머지 자동 닫힘
배송은 얼마나 걸리나요?
평균 2~3일 소요됩니다.
교환은 가능한가요?
수령 후 7일 이내 가능합니다.
해외 배송도 되나요?
현재는 국내 배송만 지원합니다.

Guidelines

✓ 권장

  • 질문은 사용자가 실제 궁금해할 문장형으로 작성
  • 기본적으로 첫 항목만 open으로 열어두기
  • 한 번에 하나만 열리게 하려면 name 속성으로 그룹핑

✕ 지양

  • 항목이 2개 이하일 때 아코디언 사용 (그냥 펼쳐서 보여주기)
  • summary 안에 링크/버튼 등 인터랙티브 요소 중첩
  • 아이콘만 두고 트리거 텍스트 생략 (스크린리더 접근성 저해)
