Textarea

기본/에러/비활성/resize 제어

Overview

Textarea는 여러 줄의 긴 텍스트를 입력받는 필드다. 한 줄짜리 짧은 입력에는 Input을 쓰고, 설명·메모·리뷰처럼 줄바꿈이 필요한 입력에 쓴다.

Variants

스타일 변형 없음 — textarea 기본형 하나만 존재한다.

Options

클래스 설명
textarea__resize-none 세로 크기 조절 비활성화. 기본은 세로로 리사이즈 가능
textarea [textarea__resize-none]

우측 하단을 드래그해서 크기를 늘릴 수 있습니다.

높이가 고정되어 늘어나지 않습니다.

States

클래스 설명
textarea__error 검증 실패. 빨간 테두리 + 힌트 텍스트 색상 변경
textarea__disabled 비활성. 회색 배경, 입력 커서 not-allowed
textarea [textarea__error] [textarea__disabled]

Guidelines

✓ 권장

  • 예상 입력 길이에 맞는 기본 높이 설정
  • 에러 시 힌트 텍스트를 구체적인 안내로 교체 (예: "10자 이상 입력해주세요")
  • 필수 안내는 label/힌트 텍스트에, placeholder는 예시로만

✕ 지양

  • 한 줄짜리 짧은 입력에 textarea 사용 (Input을 써야 함)
  • textarea__resize-none을 기본값처럼 남용
  • 에러 상태인데 원인 설명 없이 테두리 색만 바꾸기
