Dropdown Menu

드롭다운 메뉴, 체크리스트

Overview

Dropdown Menu는 트리거(아바타/버튼 등)를 눌렀을 때 나타나는 액션 목록이다. 항목을 골라 즉시 실행하는 링크형(dropdown__menu)과, 여러 개를 체크해서 필터링하는 체크리스트형(checklist) 두 가지 컨테이너가 있다. 트리거 클릭으로 여닫는 JS는 이 페이지엔 없고, 컨테이너 자체 마크업/스타일만 다룬다.

Variants

클래스 설명
dropdown__menu 링크/버튼 액션 목록. dropdown__group > dropdown__link
checklist 체크박스 목록. checklist__group > checklist__item
dropdown__menu > dropdown__group > dropdown__link vs checklist > checklist__group > checklist__item

Options

클래스 설명
dropdown__profile 선택. 메뉴 맨 위 프로필 헤더. dropdown__avatar는 생략하고 이름만 둘 수도 있음
dropdown__group 여러 개 같은 메뉴 안에서 구분선으로 그룹을 나눌 때 (예: 일반 항목 / 로그아웃)
아이콘 선택. dropdown__link 자식으로 svg를 넣으면 자동 정렬
dropdown__profile [dropdown__avatar 생략 가능]

States

항목 자체엔 hover/focus-visible 스타일만 있고 disabled 등 별도 상태 클래스는 없다. dropdown__status는 온라인 여부를 나타내는 작은 점(위 Options 데모 참고)이다.

Guidelines

✓ 권장

  • 파괴적인 액션(로그아웃, 삭제 등)은 별도 dropdown__group으로 분리
  • 체크리스트는 즉시 적용이 아니라 "적용" 버튼과 함께 쓰기
  • 아이콘은 항목 의미를 보조하는 용도로만 (아이콘만으로 판단하지 않게 텍스트 병행)

✕ 지양

  • 항목 10개 이상 나열 (검색 가능한 Dropdown 컴포넌트가 더 적합)
  • 선택형(라디오 성격)인데 체크박스로 다중 선택처럼 보이게 만들기
  • 트리거 없이 메뉴가 항상 열려있는 상태로 두기
