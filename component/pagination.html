Pagination

숫자형/텍스트형/양끝배치 등 7종

Overview

페이지네이션은 목록/테이블 결과를 여러 페이지로 나눠 이동하는 컴포넌트다. 페이지 수가 많으면 숫자+생략(…), 적으면 숫자만 노출하고, 카드뷰처럼 "페이지"라는 개념이 흐릿한 화면에는 Page X of Y 텍스트형을 쓴다. 실제 페이지 전환 JS는 붙어있지 않은 정적 데모다.

Variants

구성 설명
pagination__list 숫자형. 페이지 번호를 직접 눌러 이동
pagination__text 텍스트형. Page 1 of 10 처럼 현재/전체만 안내
pagination__list + pagination__text_none 반응형 하이브리드. 평소엔 숫자, 640px 이하에서 텍스트로 자동 전환
pagination > pagination__list 또는 pagination__text

Guidelines

✓ 권장

  • 페이지가 많으면 숫자+생략(…), 적으면(5~7개 이하) 생략 없이 전부 노출
  • "페이지" 개념이 흐릿한 화면(카드뷰 등)엔 pagination__text 사용
  • 이전/다음은 첫/마지막 페이지에서 항상 disabled 처리

✕ 지양

  • 전체 페이지 수를 모를 때 숫자형 강제 사용 (무한 스크롤이나 텍스트형이 적합)
  • 현재 페이지 표시(is-active) 없이 숫자만 나열
  • 클릭 영역을 40px 미만으로 줄이기 (터치 접근성 저하)
