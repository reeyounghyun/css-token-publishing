Switch

토글 스위치 (상태/사이즈)

Overview

스위치(Toggle)는 선택 즉시 효과가 반영되는 on/off 설정에 쓰는 컨트롤이다 (알림 켜기/끄기 등). 폼 제출이 필요한 다중 선택에는 Checkbox를 쓴다.

Variants

스타일 변형 없음 — toggle 기본형 하나만 존재한다.

Options

클래스 설명
toggle--sm 작은 크기. 기본은 md(미지정)
toggle__text > toggle__title, toggle__desc 선택. 라벨 텍스트/설명 추가, 둘 다 생략하면 스위치만 남는다
toggle > toggle__input [+ toggle__text > toggle__title [+ toggle__desc]]

States

클래스 설명
checked on 상태. 트랙 색상 + 썸이 오른쪽으로 이동
is-pressed 누르는 중. 트랙 둘레에 링 표시
is-loading 처리 중. 썸 위에 스피너, 클릭 방지(pointer-events:none)
disabled 비활성. 50% 투명도
toggle [toggle--sm] > toggle__input [is-pressed|is-loading] [checked] [disabled]

Guidelines

✓ 권장

  • 선택 즉시 반영되는 설정에만 사용 (저장 버튼이 따로 필요 없어야 함)
  • 처리 중엔 is-loading으로 클릭을 막아 중복 요청 방지
  • 라벨 텍스트는 지금 상태가 아니라 켰을 때 결과로 작성 (예: "다크 모드" O, "다크 모드 켜짐" X)

✕ 지양

  • 여러 개를 폼에 모아뒀다가 한 번에 "저장" 버튼으로 제출 (그럴 땐 Checkbox가 맞음)
  • 라벨 없이 스위치만 여러 개 나열 (무엇을 켜는지 알 수 없음)
  • 되돌릴 수 없는 파괴적 동작에 스위치 사용 (확인 절차 없이 즉시 실행됨)
