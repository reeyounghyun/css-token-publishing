Overview

카드는 관련된 정보를 하나의 컨테이너에 묶어 보여주는 컴포넌트다. 레이아웃(가로형/통계)이나 클릭 가능 여부에 따라 조합해서 쓴다.

Variants

클래스 설명
card 기본형. 세로 배치, 이미지 + 제목 + 설명 + 액션
card--horizontal 가로형. 왼쪽 썸네일 + 오른쪽 텍스트
card--stat 통계형. 숫자 지표 + 증감 표시
card [card--horizontal|card--stat]

디자인 시스템 가이드

토큰 기반으로 컴포넌트를 관리하는 방법을 정리한 문서입니다.

Options

클래스 설명
card--hoverable 선택. 카드 전체를 클릭 가능하게 만든다. <button> 태그에 붙이는 걸 전제로 함
card__footer 선택. 액션 버튼 영역, 없으면 정보만 보여주는 카드가 됨
card__placeholder 선택. 이미지/차트가 아직 없을 때 자리표시자
card card--hoverable (button 태그) / card__placeholder

주간 방문자

States

Guidelines

✓ 권장

  • 카드 안 액션은 1~2개로 제한
  • 클릭 가능한 카드는 card--hoverable + <button>으로 클릭 가능함을 명확히
  • 통계 카드는 숫자를 가장 크게, 증감은 색상+화살표로 이중 표시

✕ 지양

  • 카드 안에 카드를 중첩
  • card--hoverable 카드 내부에 또 다른 클릭 가능한 버튼/링크 넣기 (클릭 영역 충돌)
  • 통계 카드 하나에 여러 지표 욱여넣기 (카드 하나 = 지표 하나)
