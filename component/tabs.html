Tabs

Underline/Pill, disabled 탭

Overview

탭은 같은 맥락의 콘텐츠를 여러 패널로 나눠서 한 번에 하나씩만 보여주는 컴포넌트다. 이 문서 사이트 자체에서도 Button 컴포넌트의 States 섹션 등에 이미 쓰이고 있다. js/main.js의 [data-tabs] 로직이 클릭/방향키/Home/End 키보드 이동까지 처리한다.

Variants

클래스 설명
tabs 기본형. 밑줄(underline)로 활성 탭 표시
tabs--pill 세그먼트 컨트롤. 알약 모양 배경으로 활성 탭 표시

Options

독립적인 수정자 클래스는 없다.

States

속성 설명
aria-selected="true" 현재 활성 탭. tabs__panel.is-active와 함께 토글됨
disabled 선택 불가한 탭 (예: 준비중인 기능)

Guidelines

✓ 권장

  • 탭 라벨은 2단어 이내로 짧게
  • 같은 화면 안에서 동시에 볼 필요 없는 콘텐츠에만 사용
  • 준비 안 된 기능은 숨기지 말고 disabled로 존재만 알리기

✕ 지양

  • 탭 5~6개 이상 나열 (Dropdown이나 사이드 내비게이션이 더 적합)
  • 탭 전환 시 페이지 전체를 새로고침 (SPA처럼 패널만 바뀌어야 함)
  • 서로 이어지는 단계형 흐름(1단계→2단계)에 탭 사용 (그건 Stepper)
